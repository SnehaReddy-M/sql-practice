# 🛒 Online Store Database

## Project Overview

This project demonstrates the design and implementation of an Online Store Database using SQL.

The database manages customers, products, orders, and order details. It showcases table relationships, joins, aggregate functions, and business analysis queries commonly used in real-world database systems.

---

## Database Tables

- Customers
- Products
- Orders
- OrderDetails

---

## Relationships

- One Customer → Many Orders
- One Order → Many Order Details
- One Product → Many Order Details

---
